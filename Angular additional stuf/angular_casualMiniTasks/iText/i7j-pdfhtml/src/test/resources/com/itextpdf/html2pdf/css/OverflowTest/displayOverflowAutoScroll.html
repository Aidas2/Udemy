The overflow Property

The overflow property specifies whether to clip content or to add scrollbars when an element's content is too big to fit in a specified area.

  • div.ex1 { background-color: lightblue; width: 200px; height: 220px; overflow: scroll; }
  • div.ex2 { background-color: lightblue; width: 200px; height: 220px; overflow: hidden; }
  • div.ex3 { background-color: lightblue; width: 200px; height: 220px; overflow: auto; }
  • div.ex4 { background-color: lightblue; width: 200px; height: 220px; overflow: visible; }
  • span.internalScroll { background-color: magenta; overflow: scroll; }
  • span.internalAuto { background-color: magenta; overflow: auto; }
  • span.internalHidden { background-color: magenta; overflow: hidden; }
  • span.internalVisible { background-color: magenta; overflow: visible; }
