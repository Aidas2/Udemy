For all and every div element in this document 'red' is set as a value of 'color' property. However in Chrome (but not in Mozilla and Edge) 3D borders don't use 'currentColor' as according to spec, Chrome shows them in shades of gray, probably to enhance the "3D" effect. Still if specific color values are explicitly specified (not 'initial' value), these colors are applied.
